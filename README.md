# AWMK-G

AWMK-G is an all-in-one entertainment and gaming web platform that brings movies, anime, web series, K-dramas, and online games together in one place. The project focuses on providing a clean interface, smooth navigation, and a responsive design for an enjoyable user experience.

---

## Features

- Multiple entertainment categories in one platform  
- Clean and modern UI design  
- Responsive layout for desktop and mobile devices  
- Easy navigation and simple structure  
- Lightweight frontend project
